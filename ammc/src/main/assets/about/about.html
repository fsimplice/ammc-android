AMB tester software is unofficial software for testing AMB/MYLAPS sports timing devices. The software should be compatible with AMBrc Decoder, MYLAPS RC4 Decoder, AMBmx3 Decoder, TranX3 Decoder, ChipX Decoder and BIB Decoder. We have not tested with all these decoders and would be please if you have any compatibility issues to report us by email to skokys@gmail.com

Devices connected using Serial port RS-232 or USB are not supported from Android. X2 device and X2 server connection is not supported yet as well.

We recommend to use ethernet-enabled device. These connected with Ethernet network cable to a WiFi router where typical Android device can be connected. See the connection schema.

Devices connected to the same netwok with Android should be discovered automatically using UDP Broadcast that must be (normally is) enabled.

How to connect


An another application can use the P3 protocol parser on the same tablet. The application implements service that can be invoked from outside and response with parsed data Java way. If your application works on another platform, contact us for another supported platforms.
